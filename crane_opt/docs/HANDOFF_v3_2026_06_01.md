## 1. 이번 세션 코드 변경 (★중요 — 원본에 반영 필요)

### 1-1. site_loader.py
**(A) vacant_lots 파싱 추가** (신사동 세션, ~line 174 "3b" 블록)
- `spec.get("vacant_lots", [])` → `self.VACANT_LOTS` dict
- 근거: 빈 인접대지(공터·주차장)는 재실자 없음 → 그 상공 선회는 G9 공중침범 아님(소유자 동의 전제)
- F1에선 여전히 'empty'(취약성 0.5)로 집계 → 빈 대지 특성과 일치

**(B) airspace_easement 파싱 추가** (방배동 세션, ADJACENT_BUILDINGS 블록)
- 인접건물 spec에 `airspace_easement: bool` (기본 False)
- 근거: 협소대지에서 슬루반경 > 대지는 통상적. 인접 소유자 동의 하에 인접건물 '상공'으로
  지브 통과. 단 충돌없음은 height 조건으로 G2/G3에서 판정.

**(C) ALLOWED_AREA 확장** (~line 238 "7) 허용 설치 영역")
```python
road_polys    = [r["polygon"] for r in ROADS.values() if r["occupation_allowed"]]
vacant_polys  = [v["footprint"] for v in VACANT_LOTS.values()]
easement_polys= [b["footprint"] for b in ADJACENT_BUILDINGS.values() if b["airspace_easement"]]
ALLOWED_AREA  = unary_union([SITE] + road_polys + vacant_polys + easement_polys)
```
- 하위호환: vacant_lots/easement 없는 부지(공덕·역삼·신사)는 기존과 동일 (빈 리스트)

### 1-2. constraints.py
**(D) 상수 추가** (~line 76)
```python
AIRSPACE_VERT_CLEARANCE_M = 5.0   # 상공권 동의 인접건물 위 지브·인양물 수직통과 여유
                                  # [가정] 인접건물높이 + 인양물(~2m) + 안전여유(~3m)
```

**(E) G2(이격) height-aware 수정**

**(F) G3(침범) height-aware 수정**
- 동일 조건이면 그 인접건물 침범면적 집계에서 **제외**

**G7·G9는 원본 그대로 — 절대 안 건드림** (끼워맞춤 방지, 길① 결정)
