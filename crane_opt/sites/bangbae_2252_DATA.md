## 설치 방식 (핵심: 역삼·신사와 다른 유형!)
- **외부설치 + 벽체정착(월브레싱) 2단** (역삼·신사는 내부설치/코어클라이밍)
- 입면도: 마스트가 건물 **좌측 코어(계단실) 외벽에 밀착** 수직상승, 1차(~6층)·2차(~12층) 월브레싱으로 건물에 정착
- 기사: "건물 외부에 설치하여 현장 전체를 커버", "6m 도로를 활용하여 지브 조립·인양"
- → 크레인 중심 = 건물 footprint 가장자리(코어 쪽) 외부, 도로 접한 면

  배치 평면도는 기사 썸네일 이미지로만 확인됨(직접 다운로드 불가).

## 검증 결과 (길① — 정직한 적용범위 규명)
**결론: 방배동은 본 모델의 적용범위 밖 케이스. 모델을 비틀지 않고 한계를 명시한다.**

### 모델링 방침 (확정)
- 상공권 easement(인접 저층 위 지브 통과)만 반영 — 물리적 사실(높이)이므로 정당
  - site_loader: `airspace_easement` 플래그 + ALLOWED_AREA 포함
  - constraints G2/G3: height-aware (마스트 ≥ 인접건물높이+5m면 상공통과 허용)
  - 역삼·신사 무영향 확인됨 (easement=0개, knee 동일 재현)
- **G7·G9는 원본 그대로 — 손대지 않음** (끼워맞춤 방지)

### 결과 (정직한 기록)
- **옵티마이저 Pareto 62개 = 전부 내부설치** (외부설치 0개)
  - knee: (0,6) MR160C jib13 mast45 F1=101 F2=124h
- **실제 크레인 = 외부설치(월브레싱), (-6.3,0)** → 모델상 **infeasible**
  - 막은 제약: G7(외부설치 2m 이격 — 월타이 외부설치 미지원), G9(극협소대지 운용반경 초과)
- **추천(내부) vs 실제(외부)는 질적으로 불일치** (신사동 정량검증 평균 1.6m와 달리, 공법 자체가 모델 밖)

### 핵심 메시지 (논문용)
모델은 256㎡ 극협소대지에서 "내부설치만 가능"으로 판정하나, 실제 시공팀은
"외부설치+월브레싱"을 택함. 이 불일치는 모델의 가정 한계를 정직하게 드러냄:
1. 모델엔 "월타이(벽정착) 외부설치" 방식이 없음 (내부 클라이밍 or 이격 독립기초만)
2. G9가 슬루 풀서클 평면침범 기준 → 극협소대지(대지 << π·R²)에서 과도하게 보수적
→ 향후 확장: ①월타이 외부설치 분기 ②3D 충돌(높이) 기반 G9 ③러핑 운용반경 모델

### 그럼에도 방배동의 가치
- **FT-80L 제원 3중 교차검증** (도면 R20m + 기사 "20m반경" + 카탈로그)
- **모델 적용범위(validity envelope) 규명** — A+ 논문의 정직성: "되는 범위와 안 되는 범위를 안다"
- **제3의 공법(외부 월타이)** 케이스로 향후 연구방향 제시
