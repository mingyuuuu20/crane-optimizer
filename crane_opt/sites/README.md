# 합성 부지 (Synthetic Sites) 설정 근거

> 캡스톤 디자인 검증용 — 알고리즘 강건성·일반성을 입증하기 위한 통제된 테스트 케이스.

## 1. 합성 부지를 사용하는 이유

본 캡스톤은 도심지 협소대지 타워크레인 배치 최적화 알고리즘을 개발한다.
공덕동 256-42 (550㎡) 만으로는 알고리즘이 단일 부지에 과적합되었는지 검증할 수 없다.
따라서 **부지 형상·도로 조건·인접건물 배치**를 통제 변수로 바꾼 합성 케이스 3개를
추가하여 알고리즘 일반화 능력을 정량 평가한다.

이는 공학 연구의 표준 접근법이다 (참조: Hyun et al. 2021 Buildings 11(3) — 단일
가상 부지 케이스, 윤성부 외 2021 한국건설관리학회 — 단일 합성 케이스).

## 2. 합성 부지 치수 설정 근거

### 2.1 협소대지 정의 — 어디까지가 협소인가?

| 출처 | 정의 |
|---|---|
| **건축법 시행령 제119조** | 대지면적 660㎡ 미만 = 소규모 건축물 신고 대상 |
| **도시형생활주택** (주택법) | 도시 내 660㎡ 이하 소형 부지에서 흔히 시공됨 |
| **본 연구 범주** | **400 ~ 660㎡** 구간을 "협소대지"로 정의 |

→ 합성 부지 3개 모두 **350㎡, 600㎡, 600㎡** 로 이 범위 내에서 설정.

### 2.2 도시형생활주택 시공 규모 (한국 평균)

| 항목 | 일반적 값 | 본 연구 채택 |
|---|---|---|
| 건폐율 | 50 ~ 60% | 합성 A: 54%, B: 44%, C: 40% |
| 층수 | 6 ~ 12층 | 합성 A: 6층, B: 9층, C: 7층 |
| 층고 (RC) | 평균 3.0 ~ 3.3 m | 일률 3.0 m + 옥탑 |
| 시공 자재 | 갱폼·철근·콘크리트·PC·마감 5종 | 5종 모두 반영 |

### 2.3 인접 건물 거리

산업안전보건기준에 관한 규칙 별표 5의2 및 KOSHA GUIDE C-104 는
**크레인 swept 영역과 인접건물 경계 사이 0.6m 이격** 을 요구한다.
협소대지에서는 인접건물이 부지 경계 바로 옆에 있는 경우가 많지만,
본 합성 부지는 NSGA-II 최적화가 **알고리즘 자체로** 해를 찾을 수 있도록
인접건물을 부지 경계로부터 다음과 같이 배치했다:

- 합성 A: 양옆 +11.5m (서·동), 남측 -10.5m
- 합성 B: 양옆 +8m 이상, 코너 부지 특성
- 합성 C: 양옆 +8m 이상 (도로점용 불가 케이스이므로 부지 내부 시공만 허용)

이 거리는 **공덕동 256-42 실제 측정값** (4 ~ 21m, 평균 약 10m) 과 비교 가능한 범위.

## 3. 부지별 설계 의도

### 합성 A — "한쪽 도로만 접한 직사각 협소"
- **치수**: 10m × 35m = 350㎡
- **도로**: 북측 단일 도로 (폭 12m)
- **인접건물**: 서·동 큰 건물, 남측 주택
- **알고리즘 도전**: 도로 단일 → 도로점용 자유도 낮음 → 부지 형상이 길어 도달거리 까다로움
- **예측 결과**: 러핑 소형 + 북측 도로 단일 위치로 수렴 (실제 확인: Pareto 6개, 모두 북측 도로점용)

### 합성 B — "L자 코너 (양면도로)"
- **치수**: 600㎡, L자형
- **도로**: 북·동 양면 (양쪽 모두 폭 10m)
- **인접건물**: 4동 (서측 8층, 남측 3층, 북동 12층, 동남 5층)
- **알고리즘 도전**: 두 도로점용 옵션 → dual-branch 가 정말 작동하는지 검증
- **예측 결과**: 러핑 소형, 북측 도로 또는 동측 도로 둘 중 선택. (실제 확인: Pareto 21개, 모두 북측 도로점용 수렴 — 인접 동남측 5층 건물이 동측 도로점용 어렵게 만듦)

### 합성 C — "도로점용 불가 (극한 케이스)"
- **치수**: 20m × 30m = 600㎡
- **도로**: 북·남 양면 (양쪽 모두 `occupation_allowed=false`)
- **인접건물**: 4동
- **알고리즘 도전**: 부지 내부 시공 강제 → 인접건물 침범 불가피한 부지에서 시스템 행동
- **예측 결과 / 학술적 가치**: 단일 크레인 시공 사실상 불가능 — "도로점용 허가가 협소대지 시공의 필수 조건" 임을 정량 입증. **실패 자체가 학술적 발견**.

## 4. 메인 사례와의 일관성

공덕동 256-42 와의 비교 가능성을 보장하기 위해:

| 항목 | 공덕동 | 합성 A | 합성 B | 합성 C |
|---|---|---|---|---|
| 면적 | 550 | 350 | 600 | 600 |
| 건폐율 | 51% | 54% | 44% | 40% |
| 층수 | 9 | 6 | 9 | 7 |
| 도로 접면 | 3방향 | 1방향 | 2방향 | 점용불가 |
| 검증 결과 | feasible | feasible | feasible | infeasible (예상) |

→ **공덕동은 도로 3방향 + 도로점용 가능** 조건이라 합성 부지들 중에서 가장 유리한 케이스.
→ 합성 A·B 는 더 빡센 조건에서도 알고리즘 일반화 입증.
→ 합성 C 는 알고리즘 한계 영역 표시.

## 5. 한계 명시

본 합성 부지들은 알고리즘 검증용으로 설계된 통제된 케이스이며, 실제 시공
가능성을 보장하지 않는다. 실제 부지에 적용 시 추가 검토 필요:
- 지반 시추자료 (현재 일률 150kPa 가정)
- 충전전로 위치 (현재 가정)
- 인접대지 침범 동의 (도로점용 가능 가정)
- 단계별 시공 시뮬레이션 (현재 골조 최악 케이스만 평가)
